Basics

Shift + A Добавляет объект/связь
X или Delete Удаление
F3 Поиск функции
G Передвижение
S Масштаб
R Вращение
R + X/Y/Z Вращение вдоль глобальных осей
R + XX/YY/ZZ Вращение вдоль локальных осей
Двойное нажатие R Свободное вращение с трекболом
Shift(удерживать) Точное движение
Ctrl(удерживать) Последовательное движение
Shift + D Дубликат
Alt + D Дубликат связанный
H Скрыть
Alt + H Показать всё
Shift + H Скрыть всё, кроме выбранного
D(удерживать) + Левая кнопка мыши(тащить) Аннотация
D(удерживать) + Правая кнопка мыши(тащить) Стереть аннотацию

Window shortcuts

Т Панель инструментов
N Панель характеристик
Ctrl + Space Увеличить область (но оставить панель инструментов)
Ctrl + Alt + Space Полноэкранная область
Ctrl + Alt + Q Квадратный вид
Alt + Z Переключить режим X-Ray(рентген)
NumPad 7 Вид сверху
NumPad 1 Вид спереди
NumPad 3 Вид справа
Ctrl + NumPad 7 Вид снизу
Ctrl + NumPad 1 Вид сзади
Ctrl + NumPad 3 Вид слева
NumPad , Сделать объект центральным
Shift + ^ Прогулочная навигация

Change windows

Shift + F2 Видеоклип
Shift + F3 Ноды
Shift + F4 Консоль Python
Shift + F5 3D Вьюпорт
Shift + F6 График
Shift + F7 Характеристики
Shift + F8 Видео сквенсор
Shift + F9 Аутлайнер
Shift + F10 UV/Изобразение
Shift + F11 Текст
Shift + F12 Доуп Шит

General selections

Левая кнопка мыши Выбрать
A Выбрать всё
Alt + A или двойное нажатие A Убрать выделение со всего
B или левая кнопка мыши(тащить) Выбрать рамку
C Выбрать круг
Ctrl + правая кнопка мыши Выбор лассо
Ctrl + i Инвертировать выбор
Shift + L Выбрать связанные
Shift + G Выбрать похожие
Alt + левая кнопка мыши Выбрать из многих

Navigation

Колёсико мыши Вращение вокруг объекта
Shift + колёсико мыши Перемешение относительно объекта
Прокрутка или Ctrl + колёсико мыши Приближение/отдаление
Shift + ` Полёт

Object mode

Ctrl + Tab Открыть круговое меню
Tab Переключение Edit/Object mode
Ctrl + M, затем X/Y/Z (или колёсико мыши(ташить)) Открыть круговое меню
Ctrl + P Установить родителя (последний выбранный)
Alt + P Очистить родителя
Shift + Tab Переключить привязку
Alt + G Сбросить позитцию
Alt + R Сброс поворота
Alt + S Сбросить размер
Ctrl + A Применить расположение/размер/поворот
Ctrl + J Соединть выбранные объекты
Ctrl + L Скопировать атрибуты в новые объекты
Ctrl + 0/1/2/3/4/5 Добавить уровень подразделения
Alt + B Вид маски в области или очистить маску
Shift + С Переместить 3D-курсор в центр
M Переместить активный объект в колекцию
Ctrl + Alt + NumPad 0 Переместить активную камеру для просмотра
Ctrl + NumPad 0 Установить как активную камеру
